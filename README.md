# Augur Trader 

Trading with stock market behavior using historical data, technical indicators, and simulated trading. Users can analyze price charts, place paper trades, track positions, and review trade history in a risk free environment.

---

### Frontend
- React
- TypeScript
- Vite

### Backend
- FastAPI
- Python
- SQLite

---

## Features

### Market Data & Charts
- Load historical OHLCV data by ticker and date range
- Line chart and candlestick chart modes
- Zoom and pan interactions

### Stock Indicators
- SMA 
- EMA 
- MACD 

### Paper Trading

### Portfolio Tracking

### Trade Logging

Primary workflow occurs in **TradeView**, **Positions**, and **Trade Log**.
